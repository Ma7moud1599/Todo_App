# Todo App — JavaScript

A clean, browser-based Todo application built with vanilla JavaScript. Tasks persist across page reloads using localStorage.

## Features

- Add, complete, and delete tasks
- Filter tasks: All / Active / Completed
- Persistent storage via localStorage
- Task count display

## Running

Open `index.html` directly in any browser — no build step or server required.

## Tech Stack

Vanilla JavaScript, HTML5, CSS3, localStorage API
